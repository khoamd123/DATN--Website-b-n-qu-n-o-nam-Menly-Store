HƯỚNG DẪN TẠO VIRTUAL HOST CHO XAMPP

1. Mở file C:\xampp\apache\conf\extra\httpd-vhosts.conf

2. Thêm đoạn code sau vào cuối file:

<VirtualHost *:80>
    DocumentRoot "C:/xampp/htdocs/DATN_Uniclubs/public"
    ServerName uniclubs.local
    ServerAlias www.uniclubs.local
    <Directory "C:/xampp/htdocs/DATN_Uniclubs/public">
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>

3. Mở file C:\Windows\System32\drivers\etc\hosts (với quyền admin)

4. Thêm dòng này vào cuối file:
127.0.0.1 uniclubs.local

5. Restart Apache trong XAMPP Control Panel

6. Truy cập: http://uniclubs.local/login
